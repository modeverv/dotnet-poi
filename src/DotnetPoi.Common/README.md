# DotnetPoi.Common

Shared interfaces, enums, utilities, and XML writer foundation shared by all DotnetPoi packages.

> **This is a transitive dependency.** You do not need to install it directly.
> NuGet pulls it in automatically when you reference `DotnetPoi.Ooxml`, `DotnetPoi.Legacy`, or `DotnetPoi.All`.

## License

## Scope

This package contains shared API types only. User-facing Office format support lives in `DotnetPoi.Ooxml`, `DotnetPoi.Legacy`, and `DotnetPoi.Formula`.
