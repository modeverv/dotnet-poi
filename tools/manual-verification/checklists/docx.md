# docx / docm Manual Verification Checklist

- Open generated `.docx` without repair dialog.
- Open generated `.docm` without repair dialog.
- Confirm paragraphs, runs, tables, images, headers, footers, page setup, and preserved unknown content.
- Save as-is from LibreOffice.
- Reopen the saved file.
- For macro-enabled files, verify macro parts are preserved by the automated preservation checks before release.
- Record file names, LibreOffice version, screenshots, and notes in `evidence/`.
